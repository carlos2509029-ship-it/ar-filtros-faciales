# AR Filtros Faciales

Proyecto de realidad aumentada desarrollado con:

- Python
- OpenCV
- MediaPipe

## Funcionalidades

- Detección facial con FaceMesh
- Mostacho AR
- Selección de cámaras
- Compatibilidad con Camo
- Arquitectura basada en filtros

## Requisitos

### Sistema operativo

Actualmente el proyecto ha sido probado en:

- Windows 11
- PowerShell

### Versión de Python

Se recomienda utilizar:

- Python 3.10.x

Versiones más recientes de Python pueden requerir ajustes adicionales en dependencias.
